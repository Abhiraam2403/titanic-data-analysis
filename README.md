# Titanic Data Analysis 🚢

## Overview
Exploratory Data Analysis on the Titanic dataset to uncover 
factors that influenced passenger survival using Python, 
Pandas, and Seaborn.

## Dataset
- 891 passengers, 11 columns
- Source: Kaggle Titanic Dataset

## What I Did
- Data cleaning — handled missing values, dropped 
  irrelevant columns, fixed data types
- Univariate, Bivariate and Multivariate Analysis
- Outlier detection and handling using IQR method
- Feature Engineering — FamilySize, IsAlone, 
  AgeGroup, FareCategory

## Key Insights
1. Only 38% of passengers survived overall
2. Women had 74% survival rate vs 19% for men
3. 1st class passengers had 63% survival vs 24% in 3rd class
4. 96% of 1st class women survived — highest of any group
5. Passengers travelling with family survived 20% more than solo travellers
6. Children had slightly higher survival — "women and children first" 
   policy is visible directly in the data
7. Fare and survival are strongly correlated — wealthier 
   passengers had significantly better survival chances

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab

## Visualizations
- Histograms and Boxplots — distribution analysis
- Barplots — survival comparisons
- Heatmap — correlation analysis
- Pairplot — multivariate analysis
