# Titanic Data Analysis

## Overview
Exploratory Data Analysis on the Titanic dataset to find factors that influenced survival.

## Key Insights
- Only 38% of passengers survived
- Women had 74% survival rate vs 19% for men
- 1st class passengers had 63% survival vs 24% in 3rd class
- 96% of 1st class women survived
- Passengers with family had 50% survival vs 30% alone
- Children had slightly higher survival — "women and children first" policy visible in data

## Tools Used
- Python, Pandas, NumPy, Matplotlib, Seaborn

## Dataset
Titanic passenger data — 891 rows, 11 columns
